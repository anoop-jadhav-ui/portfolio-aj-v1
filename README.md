click to view the final app - https://anoopjadhav.in

###Tech stack
React Js, Firebase

### Setup

```bash
# install dependencies
$ yarn

# run dev server
$ yarn dev

# run production
$ yarn start

# build production
$ yarn build

# run unit tests
$ yarn test
```
