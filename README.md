## Setup and Commands

Use the following commands to work with this project:

```bash
# Install dependencies
$ yarn

# Run the development server
$ yarn dev

# Run the production build
$ yarn start

# Build for production
$ yarn build

# Run unit tests with coverage
$ yarn test --coverage
```
